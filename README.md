### Инструкции по установке и запуску

> ВАЖНО: требуется установленный .NET 8 SDK.

1. Клонировать репозиторий и перейти в него.
    ```bash
    git clone https://tapotheslipper/MiniPollApp.git <путь к вашей директории>
    cd <путь к вашей директории>
    ```
2. Запустить проект через .NET, IDE или любой другой Runtime.
    ```bash
    dotnet run
    ```
